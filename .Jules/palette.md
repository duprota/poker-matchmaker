## 2026-04-21 - Adding aria-labels to icon-only buttons\n**Learning:** Icon-only buttons without `aria-label` attributes are inaccessible to screen readers. This is a common pattern when using UI libraries like Shadcn UI that provide a `size="icon"` variant for buttons.\n**Action:** Ensure all icon-only buttons, especially those using `<Button size="icon">`, have a descriptive `aria-label` attribute explaining their purpose.
