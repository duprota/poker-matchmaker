## 2026-05-01 - Adding aria-labels to icon-only buttons\n**Learning:** The project relies heavily on icon-only buttons via the `<Button size="icon">` pattern from shadcn/ui. However, there is no generic automated a11y linter configured to enforce `aria-label` usage on these elements. Screen reader users would have a difficult time using the app.\n**Action:** Ensure all future implementations of icon-only buttons include descriptive `aria-label` attributes to improve screen reader accessibility.
